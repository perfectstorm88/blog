Cypher语法

[Neo4j 第三篇：Cypher查询入门](https://www.cnblogs.com/ljhdo/p/5516793.html)
```
# 设置关系
match (n)<-[r:相关治疗]-(m) set r.操作者='张三', r.prob=0.563 return n;

MATCH p=()-[r:`相关治疗`]->() RETURN p LIMIT 25
# 删除关系
match (n)<-[r:相关治疗]-(m) remove r.name return n; 
```
